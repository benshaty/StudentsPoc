## Usage

1. Clone the repo 
```bash
git clone https://github.com/benshaty/StudentsPoc.git
cd StudentsPoc
```
2. Install the dependencies.
```bash
npm install
```
3. Install and run Xampp (Apache, Mysql).
4. Setup the Database and Sample data
```bash
node .\setup\setup_script.js
```
5. Run the System
```bash
npm start
```
